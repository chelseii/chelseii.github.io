Movie Search App ‚Äì Client-Side Web Application

Browsing and exploring movie metadata with authentication and protected content

The Movie Search App is a React-based web application that allows users to search, filter, and browse metadata for films released between 1990 and 2023. Built as part of an individual university project, the focus was on designing a visually consistent, user-friendly interface while integrating secure access to protected API content using token-based authentication.

Tech Stack: JavaScript, React, Vite, Axios, React Router, CSS

  • Designed and implemented search, pagination, and protected routes using React Router and Axios, enabling users to view movie metadata and access restricted cast details upon login.
  • Built a custom, Ghibli-inspired interface using modular CSS to enhance visual consistency, accessibility, and user experience without relying on external UI libraries.
  • Structured the app with reusable components, logical routing, and a clear folder hierarchy, resulting in maintainable and scalable code.
  • Delivered a stable and responsive application that passed a majority of the core functional tests, providing a clean and intuitive user experience across key features.

Limitations: Automatic token refresh was not implemented, requiring re-login on token expiry; responsive design was partially completed; planned data visualisation features (IMDb rating trends) were excluded to prioritise core functionality.
